# Completed Backlog Archive

This folder stores completed backlog task records that no longer need to stay
in the active top-level `backlog/` queue.

Archived task files remain dependency records for `scripts/backlog_next.py` and
`scripts/backlog_blocked_next.py`. Do not claim archived tasks as executable
work. Use `scripts/backlog_archive_done.py` to move eligible completed tasks
here.
