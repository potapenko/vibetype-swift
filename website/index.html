Product features

Language and vocabulary

Say it naturally. Insert it for the work in front of you.

Translation runs only when requested. Choose a target language, then use Right Command + Right Option to transcribe and translate in one flow.

Local Dictionary entries add spelling context for names, project terms, and file names to the transcription request.

Always within reach

Start, translate, recover, and adjust from the menu bar.

If insertion fails, the accepted transcript remains available as Last Result. Paste it with Control + Command + V without replacing the macOS system clipboard.

Optional OpenAI correction is off by default. Local typography cleanup can still run without another model request.

Cost and data boundary

Clear about what stays local—and what goes to OpenAI.

HoldType has no account system, product backend, telemetry, analytics, or cloud sync. Transcription itself is an OpenAI API request made with the key you configure.

Pay for usage, not access

HoldType has no subscription. OpenAI deducts actual API usage from the OpenAI Platform account connected through your key; HoldType does not collect a monthly fee.

Why I built HoldType

Typing speed was not the problem.

The problem was how often a long prompt, review, or explanation became shorter before I finished typing it. Speaking made it easier to include the full thought.

I tried subscription tools and apps built around local and cloud model choices. They solve different problems. In my daily work, OpenAI transcription gave me the first pass I trusted most often.

I wanted a narrower setup: one provider, a sensible default, direct API billing, and no product subscription to remember. HoldType is built and tested through the same voice-first workflow it is meant to support.

New to API keys?

Create one in a couple of minutes.

An API key is a private credential that lets HoldType send audio for transcription through your OpenAI Platform account. It is separate from your ChatGPT login and subscription.

  1. Sign in to OpenAI Platform and open the API key page.
  2. Create a secret key and copy it when OpenAI shows it.
  3. In HoldType, open Settings → OpenAI, paste the key, and save.
Create an OpenAI key Official instructions

OpenAI shows the full key only when it is created. Keep it private and paste it only into HoldType, which stores it locally in macOS Keychain. API billing is separate from ChatGPT, and a new OpenAI Platform account may require prepaid credit.

Frequently asked questions

Short answers to the decisions that matter before installing.

Why is an OpenAI API key required?

HoldType sends transcription requests through your own OpenAI Platform account. The key is stored locally in macOS Keychain. If API keys are new to you, follow the three-step guide above.

Is ChatGPT Plus enough?

No. ChatGPT subscriptions and OpenAI Platform API billing are separate. HoldType needs an API key with OpenAI Platform billing configured.

What does dictation cost?

HoldType has no fixed fee. OpenAI deducts API usage from your OpenAI Platform balance, and new API accounts may require prepaid credit. HoldType’s Billing screen estimates successful transcriptions on this Mac; optional correction and translation are separate requests.

What data is sent to OpenAI?

Audio is sent for transcription. Optional correction and translation send transcript text in separate requests, and optional nearby context sends a short excerpt with the transcription request.

Is audio stored?

Completed audio is not retained by default. A recoverable failure may keep bounded session-only audio for Retry, and optional recording-cache retention is local and user-controlled.

Which Mac apps work?

HoldType inserts accepted text in most Mac apps through macOS Accessibility support. If handoff fails, Last Result remains available for recovery.

Which languages are supported?

Transcription can use automatic or fixed source-language settings. Translation uses the target language you configure in HoldType.

Why does HoldType need macOS permissions?

Microphone access records speech, Input Monitoring supports the global shortcut, and Accessibility enables text insertion into the active app.

Is HoldType open source?

HoldType is source-available under FSL 1.1 with an MIT future license. Each released version converts to MIT two years after release.
